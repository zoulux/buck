  • public class ThriftArtifactCacheProtocol
extends Object
    All messages generate by this Protocol will be in the following binary format: - int32 Big Endian size bytes of thrift serialized thriftData. - Thrift serialized thriftData. - Remainder of the stream contains binary payload data. Information about it is available in the Thrift thriftData.
