        public BuildExecutionResult executeBuild(Iterable<? extends HasBuildTarget> targetish,
                                boolean isKeepGoing)
                                  throws IOException,
                                         StepFailedException,
                                         ExecutionException,
                                         InterruptedException
        If isKeepGoing is false, then this returns a future that succeeds only if all of rulesToBuild build successfully. Otherwise, this returns a future that should always succeed, even if individual rules fail to build. In that case, a failed build rule is indicated by a null value in the corresponding position in the iteration order of rulesToBuild.
        Parameters:
        targetish - The targets to build. All targets in this iterable must be unique.
