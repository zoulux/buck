Class DummyRDotJava

  • java.lang.Object
    • com.facebook.buck.rules.AbstractBuildRule

  • All Implemented Interfaces:
    HasJavaAbi, HasBuildTarget, BuildRule, AbiRule, Comparable<BuildRule>

    public class DummyRDotJava
extends AbstractBuildRule
implements AbiRule, HasJavaAbi
    Buildable that takes in a list of HasAndroidResourceDeps and for each of these rules, first creates an R.java file using MergeAndroidResourcesStep and compiles it to generate a corresponding R.class file. These are called "dummy" R.java files since these are later merged together into a single R.java file by AaptStep.
