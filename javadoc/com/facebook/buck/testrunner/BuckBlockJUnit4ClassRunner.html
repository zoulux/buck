    public class BuckBlockJUnit4ClassRunner
extends BlockJUnit4ClassRunner
    JUnit-4-compatible test class runner that supports the concept of a "default timeout." If the value of defaultTestTimeoutMillis passed to the constructor is non-zero, then it will be used in place of Test.timeout() if Test.timeout() returns zero.

    The superclass, BlockJUnit4ClassRunner, was introduced in JUnit 4.5 as a published API that was designed to be extended.

    This runner also creates Descriptions that allow JUnitRunner to filter which test-methods to run and should be forced into the test code path whenever test-selectors are in use.
