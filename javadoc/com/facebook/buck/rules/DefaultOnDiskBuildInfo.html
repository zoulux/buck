    OnDiskBuildInfo

    public class DefaultOnDiskBuildInfo
extends Object
implements OnDiskBuildInfo
    Utility for reading the metadata associated with a build rule's output. This is metadata that would have been written by a BuildInfoRecorder when the rule was built initially.

    Such metadata is stored as key/value pairs.
