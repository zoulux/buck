com.facebook.buck.rules

Class BuildInfoRecorder

  • java.lang.Object
    • com.facebook.buck.rules.BuildInfoRecorder

  • public class BuildInfoRecorder
extends Object
    Utility for recording the paths to the output files generated by a build rule, as well as any metadata about those output files. This data will be packaged up into an artifact that will be stored in the cache. The metadata will also be written to disk so it can be read on a subsequent build by an OnDiskBuildInfo.
