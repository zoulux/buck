com.facebook.buck.httpserver

Class WebServer

  • java.lang.Object
    • com.facebook.buck.httpserver.WebServer

  • public class WebServer
extends Object
    A WebSocket server that reports events of buck. The WebServer can be modeled as: a) a network listener/dispatcher which sits on the port and listens for incoming http requests. b) static handlers - list of handlers that are bound to the current process (list of traces/logs/socket), c) 'normal' handlers, bound to the daemon state. We would like the server to go down as infrequently as possible (this makes it simpler for /ws users who would otherwise need to reconnect and potentially lose state) which is the reason for section c) above.
