com.facebook.buck.parser

Class Parser

  • java.lang.Object
    • com.facebook.buck.parser.Parser

  • public class Parser
extends Object
    High-level build file parsing machinery. Primarily responsible for producing a TargetGraph based on a set of targets. Caches build rules to minimise the number of calls to python and processes filesystem WatchEvents to invalidate the cache as files change.
