        public WorkspaceAndProjectGenerator setGroupableTests(Set<TargetNode<AppleTestDescription.Arg>> tests)
        Set the tests that can be grouped. These tests will always be generated as static libraries, and linked into synthetic test targets. While it may seem unnecessary to do this for tests which may not be able to share a bundle with any other test, note that WorkspaceAndProjectGenerator only has a limited view of all the tests that exists, but the generated projects are shared amongst all Workspaces.
