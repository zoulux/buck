Class RepackZipEntriesStep

    Step

    public class RepackZipEntriesStep
extends Object
implements Step
    A command that creates a copy of a ZIP archive, making sure that certain user-specified entries are packed with a certain compression level. Can be used, for instance, to force the resources.arsc file in an Android .apk to be compressed.
