com.facebook.buck.event

Class EventTime

  • java.lang.Object
    • com.facebook.buck.event.EventTime

  • public class EventTime
extends Object
    Utility class to track cumulative times (wall clock and if supported, user/system CPU times) of an event which may be discontinuous. The event must start and end on the same thread; however, it may subsequently start on a different thread. Note that wall time is tracked in milliseconds, but CPU time is tracked in nanoseconds. This is to minimize impedance mismatch with existing APIs.
