com.facebook.buck.util.hash

Class AppendingHasher

  • java.lang.Object
    • com.facebook.buck.util.hash.AppendingHasher
  • All Implemented Interfaces:
    com.google.common.hash.Hasher, com.google.common.hash.PrimitiveSink

    @NotThreadSafe
public class AppendingHasher
extends Object
implements com.google.common.hash.Hasher
    Hasher whose put* calls are forwarded to a sequence of Hasher objects.

    When hash() is invoked, the hash() method of the first Hasher in the sequence is invoked, and then the Hasher is removed from the sequence. This makes it possible to invoke additional put* methods after hash() is invoked, which is not generally true of most implementations of Hasher. Example: 

     Hasher appendingHasher = new AppendingHasher(Hashing.sha1(), 2);
 appendingHasher.putInt(42);
 HashCode hashOf42 = appendingHasher.hash();

 // This call would fail if appendingHasher were created via Hashing.sha1().newHasher().
 appendingHasher.putInt(24);
 HashCode hashOf42And24 = appendingHasher.hash();
