        Launches then executes a process with the specified params.

        If options contains ProcessExecutor.Option.PRINT_STD_OUT, then the stdout of the process will be written directly to the stdout passed to the constructor of this executor. Otherwise, the stdout of the process will be made available via ProcessExecutor.Result.getStdout().

        If options contains ProcessExecutor.Option.PRINT_STD_ERR, then the stderr of the process will be written directly to the stderr passed to the constructor of this executor. Otherwise, the stderr of the process will be made available via ProcessExecutor.Result.getStderr().

      • execute

        public ProcessExecutor.Result execute(Process process,
                             Set<ProcessExecutor.Option> options,
                             com.google.common.base.Optional<String> stdin,
                             com.google.common.base.Optional<Long> timeOutMs,
                             com.google.common.base.Optional<com.google.common.base.Function<Process,Void>> timeOutHandler)
                               throws InterruptedException
        Executes the specified already-launched process.

        If options contains ProcessExecutor.Option.PRINT_STD_OUT, then the stdout of the process will be written directly to the stdout passed to the constructor of this executor. Otherwise, the stdout of the process will be made available via ProcessExecutor.Result.getStdout().

        If options contains ProcessExecutor.Option.PRINT_STD_ERR, then the stderr of the process will be written directly to the stderr passed to the constructor of this executor. Otherwise, the stderr of the process will be made available via ProcessExecutor.Result.getStderr().

        Parameters:
        timeOutHandler - If present, this method will be called before the process is killed.
        Throws:
        InterruptedException
