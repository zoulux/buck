Class DebugPathSanitizer

  • java.lang.Object
    • com.facebook.buck.cxx.DebugPathSanitizer

  • public class DebugPathSanitizer
extends Object
    Encapsulates all the logic to sanitize debug paths in native code. Currently, this just supports sanitizing the compilation directory that compilers typically embed in binaries when including debug sections (e.g. using "-g" with gcc/clang inserts the compilation directory in the DW_AT_comp_dir DWARF field).
