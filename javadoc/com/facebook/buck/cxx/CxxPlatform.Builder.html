Class CxxPlatform.Builder

  • java.lang.Object
    • com.facebook.buck.cxx.CxxPlatform.Builder
  • Enclosing class:
    CxxPlatform

    @NotThreadSafe
public static final class CxxPlatform.Builder
extends Object
    Builds instances of type CxxPlatform. Initialize attributes and then invoke the build() method to create an immutable instance.

    Builder is not thread-safe and generally should not be stored in a field or collection, but instead used immediately to create instances.
