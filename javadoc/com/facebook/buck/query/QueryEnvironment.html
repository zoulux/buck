Interface QueryEnvironment

  • All Known Implementing Classes:
    BuckQueryEnvironment

    public interface QueryEnvironment
    The environment of a Buck query that can evaluate queries to produce a result. The query language is documented at docs/command/query.soy

      • buildTransitiveClosure

        void buildTransitiveClosure(Set<QueryTarget> targetNodes,
                          int maxDepth,
                          com.google.common.util.concurrent.ListeningExecutorService executor)
                            throws InterruptedException,
                                   QueryException
        Construct the dependency graph for a depth-bounded forward transitive closure of all nodes in "targetNodes". The identity of the calling expression is required to produce error messages.

        If a larger transitive closure was already built, returns it to improve incrementality, since all depth-constrained methods filter it after it is built anyway.

        Throws:
        InterruptedException
        QueryException
