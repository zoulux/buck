Class OCamlBuildContext.Builder

  • java.lang.Object
    • com.facebook.buck.ocaml.OCamlBuildContext.Builder
  • Enclosing class:
    OCamlBuildContext

    @NotThreadSafe
public static final class OCamlBuildContext.Builder
extends Object
    Builds instances of type OCamlBuildContext. Initialize attributes and then invoke the build() method to create an immutable instance.

    Builder is not thread-safe and generally should not be stored in a field or collection, but instead used immediately to create instances.
