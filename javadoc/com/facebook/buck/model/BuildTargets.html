        Return a path to a file in the buck-out/bin/ directory. format will be prepended with the BuckPaths.getScratchDir() and the target base path, then formatted with the target short name.
        Parameters:
        target - The BuildTarget to scope this path to.
        format - String.format(java.lang.String, java.lang.Object...) string for the path name. It should contain one "%s", which will be filled in with the rule's short name. It should not start with a slash.
        Returns:
        A Path under buck-out/bin, scoped to the base path of target.
