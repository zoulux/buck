Class Either<LEFT,RIGHT>

  • java.lang.Object
    • com.facebook.buck.model.Either<LEFT,RIGHT>

  • public final class Either<LEFT,RIGHT>
extends Object
    A simple discriminated union of two parameters. Note that the corresponding coercer #{link EitherTypeCoercer} is left-biased. That is, if the input value can coerce to the left parameter, then the left value will be populated, even if it can also be coerced to the right parameter.
