      • shouldUseJsonFormat

        public boolean shouldUseJsonFormat()
        Determines if the results should be in JSON format. This is true either when explicitly required by the --json argument or when there is at least one item in the --output-attributes arguments.

        The --output--attributes arguments implicitly enables JSON format because there is currently no way to output attributes in non-JSON format. Also, it keeps this command consistent with the query command.
