    public class DefaultJavaLibrary
extends AbstractBuildRule
implements JavaLibrary, HasClasspathEntries, ExportDependencies, InitializableFromDisk<JavaLibrary.Data>, AndroidPackageable, SupportsInputBasedRuleKey, SupportsDependencyFileRuleKey, JavaLibraryWithTests
    Suppose this were a rule defined in src/com/facebook/feed/BUCK: 
     java_library(
   name = 'feed',
   srcs = [
     'FeedStoryRenderer.java',
   ],
   deps = [
     '//src/com/facebook/feed/model:model',
     '//third-party/java/guava:guava',
   ],
 )
    Then this would compile FeedStoryRenderer.java against Guava and the classes generated from the //src/com/facebook/feed/model:model rule.
