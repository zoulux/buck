    public class AnnotationProcessingParams

implements RuleKeyAppendable
    Information for annotation processing. Annotation processing involves a set of processors, their classpath(s), and a few other command-line options for javac. We want to be able to specify all this various information in a BUCK configuration file and use it when we generate the javac command. This facilitates threading the information through buck in a more descriptive package rather than passing all the components separately.
